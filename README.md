# RUN ANALYZER PRO

RUN ANALYZER PRO adalah aplikasi desktop berbasis Python yang digunakan untuk menganalisis performa lari pengguna.

## Deskripsi
Aplikasi ini membantu pengguna menghitung pace, kecepatan rata-rata, dan estimasi kalori terbakar berdasarkan jarak, waktu, dan berat badan. Selain itu, aplikasi menyediakan fitur target jarak harian, progres lari, rekomendasi nutrisi, jadwal latihan mingguan, serta riwayat aktivitas lari.

## Fitur Utama
- Perhitungan pace, kecepatan, dan kalori
- Target dan progres jarak harian
- Rekomendasi nutrisi
- Jadwal latihan mingguan
- Riwayat aktivitas lari
- Mode gelap dan terang

## Teknologi
- Python
- Tkinter (GUI)

## Cara Menjalankan
1. Pastikan Python sudah terinstal  
2. Jalankan file **run_analyzer_pro.py**
